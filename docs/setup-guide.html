Foxentry Data Cleaner · setup

Setup guide

Getting your Foxentry API key  ·  for the data cleaner  ·  API: Foxentry v2.1
Before your first run you need a Foxentry account and an API project with an API key. This tool talks to the Foxentry REST API, so the steps below take only a few minutes.
💻 Running it: the recommended Windows build is a signed single‑exe — no Python or pip install needed; just run FoxentryDataCleaner.exe. Its data (config.env, input/, output/, logs/) is created next to the exe, so keep it in a user‑writable folder. You can verify the Authenticode signature (issuer AVANTRO s.r.o.), the published SHA256, and the build provenance attestation before running. Developers/auditors can still run from source with START.bat / python run.py.
1
Create an account
Sign up at app.foxentry.com. Every new project includes a free 14-day trial.
2
Create an API project
Click the orange New project button, choose Application / API (not Web / JS — this tool uses the REST API), then Continue. Fill in a name and the project country and finish.
3
Generate an API key
Open the project → Settings → tab API keys → click the empty Create API key card. Name it (e.g. “Data cleaner”), optionally set an expiration or IP restriction, then click Create new key.
4
Copy the key immediately
For security Foxentry shows the full key only once, right after creating it — later you’ll see just part of it. Copy it now; if you lose it you have to generate a new one.
5
Make sure you have credit
Validation consumes credits. During the 14-day trial you can test for free; afterwards top up the tariff/credit in the project (API → Top up tariff / credit).
6
Paste the key into the cleaner
In the cleaner’s last step (Order) paste the key into the API key field — it’s stored locally and shown masked. Alternatively put FOXENTRY_API_KEY=… into config.env.
💡 Web project vs. API project: Foxentry also offers a Web / JS project for live autocomplete in website forms. This tool is different — it cleans existing files via the REST API, so always pick an Application / API project. Full manual: Project types · Generating an API key.
⏳ Just created the key? A brand-new key can take a short moment to activate on Foxentry’s side. If the first run reports an invalid key, the cleaner automatically waits ~20 s and retries once.
Need help? See the in-app Manual or contact Foxentry support.

Návod na zprovoznění

Získání API klíče Foxentry  ·  pro čistič dat  ·  API: Foxentry v2.1
Než spustíte první validaci potřebujete účet Foxentry a API projekt s API klíčem. Nástroj komunikuje s REST API Foxentry, takže následující kroky zaberou jen pár minut.
💻 Spuštění: doporučená verze pro Windows je podepsaný single‑exe — není potřeba instalovat Python ani pip, stačí spustit FoxentryDataCleaner.exe. Data (config.env, input/, output/, logs/) vznikají vedle exe, proto ho mějte ve složce, kam smíte zapisovat. Před spuštěním lze ověřit Authenticode podpis (vydavatel AVANTRO s.r.o.), zveřejněný SHA256 i atestaci původu buildu. Vývojáři/auditoři mohou stále spouštět ze zdroje přes START.bat / python run.py.
1
Založte si účet
Zaregistrujte se na app.foxentry.com. Ke každému novému projektu dostanete 14denní zkušební verzi zdarma.
2
Vytvořte API projekt
Klikněte na oranžové tlačítko Nový projekt, zvolte Aplikace / API (ne Webový formulář / JS — tento nástroj jede přes REST API) a dejte Pokračovat. Vyplňte název a zemi projektu a dokončete.
3
Vygenerujte API klíč
Otevřete projekt → Nastavení → záložka API klíče → klikněte na prázdnou kartu Vytvořit API klíč. Pojmenujte ho (např. „Čistič dat“), volitelně nastavte expiraci nebo omezení na IP a klikněte na Vytvořit nový klíč.
4
Klíč si hned zkopírujte
Z bezpečnostních důvodů Foxentry zobrazí celý klíč jen jednou, ihned po vytvoření — později uvidíte už jen jeho část. Zkopírujte si ho teď; při ztrátě musíte vygenerovat nový.
5
Mějte kredit
Validace spotřebovává kredit. Během 14denní zkušební verze testujete zdarma; poté dobijte tarif/kredit v projektu (API → Dobíjení tarifu / kreditu).
6
Vložte klíč do čističe
V posledním kroku čističe (Objednávka) vložte klíč do pole API klíč — uloží se lokálně a zobrazuje se maskovaně. Případně vložte FOXENTRY_API_KEY=… do souboru config.env.
💡 Webový projekt vs. API projekt: Foxentry nabízí i Webový / JS projekt pro našeptávání přímo ve formulářích webu. Tento nástroj je jiný — čistí existující soubory přes REST API, takže vždy zvolte projekt Aplikace / API. Celý manuál: Typy projektů · Vygenerování API klíče.
⏳ Klíč jste právě vytvořili? Čerstvý klíč může chvíli trvat, než se na straně Foxentry aktivuje. Pokud první běh nahlásí neplatný klíč, čistič automaticky počká ~20 s a jednou to zopakuje.
Potřebujete pomoct? Mrkněte na Manuál v aplikaci nebo se ozvěte podpoře Foxentry.
